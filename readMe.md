
# how to start application

### install python to PC

### install docker desktop or docker

### в консолі команда docker run -d --name fastapi -e MONGO_INITDB_ROOT_USERNAME=admin -e MONGO_INITDB_ROOT_PASSWORD=password123 -p 27017:27017 mongo in terminal

### в консолі . .\venv\Scripts\activate

### в консолі команда -  uvicorn app.main:app --reload 


### в консолі іншій cd app
### і потім в тій же новій консолі команда streamlit run app.py
